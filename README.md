# Number Guessing Game

A beginner-friendly Python mini project where the player guesses a randomly generated number between 1 and 100.

## Features
- Random number generation
- Unlimited attempts
- Hints (Too High / Too Low)
- Counts the number of attempts

## Technologies Used
- Python

## How to Run

```bash
python Number-Guessing-Game.py
```
